JavaScript-Lexikon
  • console.log(): Gibt eine Nachricht in der Entwicklerkonsole aus.
  • alert(): Zeigt eine Benachrichtigung mit einer Nachricht an.
  • prompt(): Zeigt einen Dialog zur Eingabe von Daten an.
  • confirm(): Zeigt einen Bestätigungsdialog mit OK/Abbrechen-Optionen an.
  • document.getElementById(): Ruft ein HTML-Element über seine ID ab.
  • document.querySelector(): Ruft das erste passende HTML-Element über einen CSS-Selektor ab.
  • addEventListener(): Fügt einen Ereignislistener zu einem Element hinzu.
  • function functionName(): Definiert eine benannte JavaScript-Funktion.
  • if-else: Führt Code aus, wenn eine Bedingung wahr ist, andernfalls wird alternativer Code ausgeführt.
  • for loop: Führt Code in einer Schleife eine bestimmte Anzahl von Malen aus.
  • while loop: Führt Code in einer Schleife aus, solange eine Bedingung wahr ist.
  • switch: Auswahlanweisung, die verschiedene Codezweige basierend auf einem Ausdruck ausführt.
  • typeof: Ermittelt den Datentyp eines Werts.
  • parseInt(): Konvertiert eine Zeichenkette in eine ganze Zahl.
  • parseFloat(): Konvertiert eine Zeichenkette in eine Fließkommazahl.
  • String(): Konvertiert einen Wert in eine Zeichenkette.
  • Number(): Konvertiert einen Wert in eine Zahl.
  • Array: Ein JavaScript-Objekt zur Speicherung von Listen von Werten.
  • Object: Ein generisches JavaScript-Objekt.
  • JSON.parse(): Wandelt eine JSON-Zeichenkette in ein JavaScript-Objekt um.
  • JSON.stringify(): Wandelt ein JavaScript-Objekt in eine JSON-Zeichenkette um.
  • this: Verweist auf das aktuelle Objekt oder den aktuellen Kontext.
  • new: Erstellt eine Instanz eines Objekts oder einer Konstruktorfunktion.
  • return: Gibt einen Wert aus einer Funktion zurück.
  • typeof: Ermittelt den Datentyp eines Werts.
  • instanceof: Überprüft, ob ein Objekt eine Instanz einer bestimmten Klasse oder eines Konstruktors ist.
  • Math.random(): Gibt eine zufällige Fließkommazahl zwischen 0 (inklusive) und 1 (exklusive) zurück.
  • Math.floor(): Rundet eine Fließkommazahl auf die nächstkleinere ganze Zahl ab.
  • Math.ceil(): Rundet eine Fließkommazahl auf die nächsthöhere ganze Zahl auf.
  • Math.round(): Rundet eine Fließkommazahl auf die nächstgelegene ganze Zahl.
  • Math.min(): Gibt den kleinsten Wert von angegebenen Zahlen zurück.
  • Math.max(): Gibt den größten Wert von angegebenen Zahlen zurück.
  • Math.abs(): Gibt den absoluten Wert einer Zahl zurück.
  • Math.pow(): Gibt eine Zahl hoch eine Exponentenzahl zurück.
  • Math.sqrt(): Gibt die Quadratwurzel einer Zahl zurück.
  • Math.random(): Gibt eine zufällige Fließkommazahl zwischen 0 (inklusive) und 1 (exklusive) zurück.
  • Date: Ein JavaScript-Objekt zur Arbeit mit Datums- und Zeitwerten.
  • new Date(): Erstellt eine neue Instanz des Date-Objekts, das das aktuelle Datum und die Uhrzeit enthält.
  • Date.now(): Gibt die Anzahl der Millisekunden seit dem 1. Januar 1970 (UNIX-Zeit) zurück.
  • String.prototype.length: Gibt die Länge einer Zeichenkette zurück.
  • String.prototype.indexOf(): Gibt den Index des ersten Vorkommens einer Zeichenkette in einer anderen Zeichenkette zurück.
  • String.prototype.slice(): Extrahiert einen Teil einer Zeichenkette und gibt sie als neue Zeichenkette zurück.
  • String.prototype.substr(): Extrahiert eine angegebene Anzahl von Zeichen aus einer Zeichenkette, beginnend an einem bestimmten Index.
  • String.prototype.replace(): Ersetzt Teil einer Zeichenkette durch eine andere Zeichenkette.
  • String.prototype.toLowerCase(): Konvertiert eine Zeichenkette in Kleinbuchstaben.
  • String.prototype.toUpperCase(): Konvertiert eine Zeichenkette in Großbuchstaben.
  • String.prototype.split(): Teilt eine Zeichenkette in ein Array von Teilzeichenketten auf.
  • Array.prototype.length: Gibt die Länge eines Arrays zurück.
  • Array.prototype.push(): Fügt Elemente am Ende eines Arrays hinzu.
  • Array.prototype.pop(): Entfernt das letzte Element eines Arrays und gibt es zurück.
  • Array.prototype.shift(): Entfernt das erste Element eines Arrays und gibt es zurück.
  • Array.prototype.unshift(): Fügt Elemente am Anfang eines Arrays hinzu.
  • Array.prototype.slice(): Extrahiert eine Teilmenge von Elementen aus einem Array und gibt sie als neues Array zurück.
  • Array.prototype.splice(): Entfernt, ersetzt oder fügt Elemente an beliebiger Position in einem Array ein.
  • Array.prototype.join(): Verbindet die Elemente eines Arrays zu einer Zeichenkette.
  • Array.prototype.indexOf(): Gibt den Index des ersten Vorkommens eines Elements in einem Array zurück.
  • Array.prototype.lastIndexOf(): Gibt den Index des letzten Vorkommens eines Elements in einem Array zurück.
  • Array.prototype.forEach(): Führt eine Funktion für jedes Element in einem Array aus.
  • Array.prototype.map(): Erzeugt ein neues Array, indem eine Funktion auf jedes Element im ursprünglichen Array angewendet wird.
  • Array.prototype.filter(): Erzeugt ein neues Array mit Elementen, die eine angegebene Bedingung erfüllen.
  • Array.prototype.reduce(): Reduziert die Elemente eines Arrays auf einen einzigen Wert mit einer akkumulierten Funktion.
  • Array.prototype.sort(): Sortiert die Elemente eines Arrays.
  • Array.prototype.reverse(): Dreht die Reihenfolge der Elemente eines Arrays um.
  • Array.isArray(): Überprüft, ob ein Wert ein Array ist.
  • Object.keys(): Gibt ein Array zurück, das die Eigenschaftennamen eines Objekts enthält.
  • Object.values(): Gibt ein Array zurück, das die Werte der Eigenschaften eines Objekts enthält.
  • Object.entries(): Gibt ein Array von Arrays zurück, die jeweils ein Schlüssel-Wert-Paar eines Objekts darstellen.
  • Object.prototype.hasOwnProperty(): Überprüft, ob ein Objekt eine bestimmte Eigenschaft direkt enthält.
  • Object.prototype.toString(): Gibt eine Zeichenkette zurück, die den Typ eines Objekts angibt.
  • Object.assign(): Kombiniert Eigenschaften von einem oder mehreren Quellobjekten in ein Zielobjekt.
  • Object.create(): Erstellt ein neues Objekt mit dem angegebenen Prototyp.
  • Object.freeze(): Verhindert Änderungen an einem Objekt, indem seine Eigenschaften schreibgeschützt gemacht werden.
  • Object.seal(): Verhindert das Löschen von Eigenschaften eines Objekts, aber ermöglicht das Ändern ihrer Werte.
  • Object.getPrototypeOf(): Gibt den Prototyp eines Objekts zurück.
  • Object.setPrototypeOf(): Legt den Prototyp eines Objekts fest.
  • setTimeout(): Führt eine Funktion einmalig nach einer bestimmten Verzögerung aus.
  • setInterval(): Führt eine Funktion in regelmäßigen Abständen aus.
  • clearTimeout(): Bricht eine mit setTimeout() geplante Ausführung ab.
  • clearInterval(): Bricht eine mit setInterval() geplante Ausführung ab.
  • localStorage: Ein JavaScript-Objekt zur Speicherung von Schlüssel-Wert-Paaren lokal im Browser.
  • sessionStorage: Ein JavaScript-Objekt zur temporären Speicherung von Schlüssel-Wert-Paaren im Browser während der Sitzung.
  • XMLHttpRequest: Ein JavaScript-Objekt zur Durchführung von HTTP-Anforderungen.
  • fetch(): Ermöglicht asynchrone HTTP-Anfragen und den Umgang mit den Antwortdaten als Promises.
  • try-catch: Ermöglicht das Abfangen und Behandeln von Ausnahmen (Fehler) in JavaScript.
  • throw: Löst eine eigene Ausnahme aus.
  • Promise: Ein JavaScript-Objekt, das die asynchrone Verarbeitung von Daten repräsentiert und den Umgang mit zukünftigen Werten ermöglicht.
  • async/await: Ein Konstrukt in JavaScript, das die asynchrone Verarbeitung mit Promises vereinfacht.
