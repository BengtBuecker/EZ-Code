CSS-Lexikon
  • text-decoration: Legt die Dekoration von Text fest (z. B. underline, line-through).
  • text-transform: Ändert die Groß-/Kleinschreibung von Text.
  • text-shadow: Fügt einen Schatten um den Text hinzu.
  • font-weight: Definiert die Schriftstärke (z. B. normal, bold).
  • font-style: Setzt den Schriftschnitt (z. B. normal, italic).
  • letter-spacing: Bestimmt den Abstand zwischen Buchstaben.
  • word-spacing: Definiert den Abstand zwischen Wörtern.
  • line-height: Legt die Zeilenhöhe des Textes fest.
  • text-align: Bestimmt die horizontale Ausrichtung des Textes.
  • vertical-align: Setzt die vertikale Ausrichtung von Inline-Elementen.
  • white-space: Kontrolliert den Umgang mit Leerzeichen im Text.
  • list-style-type: Definiert das Aufzählungszeichen von Listen.
  • list-style-position: Legt die Position des Aufzählungszeichens fest.
  • list-style-image: Verwendet ein Bild als Aufzählungszeichen.
  • box-sizing: Steuert, wie das Boxmodell berechnet wird.
  • position: Definiert die Positionierung eines Elements.
  • top, right, bottom, left: Steuert die Position eines positionierten Elements.
  • display: Bestimmt das Layoutverhalten eines Elements.
  • visibility: Kontrolliert die Sichtbarkeit eines Elements.
  • opacity: Setzt die Transparenz eines Elements.
  • background-image: Fügt ein Hintergrundbild hinzu.
  • background-repeat: Legt fest, wie das Hintergrundbild wiederholt wird.
  • background-position: Setzt die Position des Hintergrundbildes.
  • background-size: Definiert die Größe des Hintergrundbildes.
  • background-attachment: Bestimmt, ob das Hintergrundbild mit dem Element scrollt.
  • border-style: Definiert den Stil des Rahmens.
  • border-width: Legt die Breite des Rahmens fest.
  • border-color: Setzt die Farbe des Rahmens.
  • border-radius: Erzeugt abgerundete Ecken am Rahmen.
  • box-shadow: Fügt einen Schatten um eine Box hinzu.
  • margin: Steuert den äußeren Abstand eines Elements.
  • margin-top, margin-right, margin-bottom, margin-left: Steuert den Abstand für jede Seite des Elements.
  • padding: Steuert den inneren Abstand eines Elements.
  • padding-top, padding-right, padding-bottom, padding-left: Steuert den Abstand für jede Seite des inneren Elements.
  • width: Legt die Breite eines Elements fest.
  • height: Setzt die Höhe eines Elements.
  • max-width: Begrenzt die maximale Breite eines Elements.
  • max-height: Begrenzt die maximale Höhe eines Elements.
  • min-width: Setzt die minimale Breite eines Elements.
  • min-height: Setzt die minimale Höhe eines Elements.
  • float: Elemente können links oder rechts im Layout schweben.
  • clear: Legt fest, auf welcher Seite eines schwebenden Elements kein anderes Element erscheinen darf.
  • overflow: Kontrolliert, wie mit Inhalten umgegangen wird, die über den sichtbaren Bereich hinausgehen.
  • cursor: Ändert das Aussehen des Mauszeigers, wenn er über einem Element schwebt.
  • z-index: Legt die Stapelreihenfolge von Elementen fest.
  • text-indent: Setzt den Einzug für den ersten Buchstaben eines Elements.
  • text-overflow: Bestimmt, wie Text abgeschnitten wird, wenn er nicht in ein Element passt.
  • word-wrap: Steuert, ob lange Wörter umgebrochen werden können, um das Layout nicht zu zerstören.
  • text-justify: Definiert die Ausrichtung und den Zeilenabstand für den Text in einem Textblock.
  • transform: Führt eine Transformation (Skalierung, Drehung usw.) auf einem Element durch.
  • transition: Erzeugt einen fließenden Übergang zwischen verschiedenen CSS-Eigenschaften.
  • animation: Fügt eine Animation auf einem Element hinzu.
  • @media: Definiert die Regeln für verschiedene Medientypen (Bildschirmgrößen).
  • @keyframes: Definiert den Verlauf einer Animation mit verschiedenen Schlüsselbildern.
  • :hover: Definiert die Stile, wenn der Mauszeiger über einem Element schwebt.
  • :active: Definiert die Stile, wenn ein Element aktiviert wird (z. B. Mausklick).
  • :focus: Definiert die Stile, wenn ein Element den Fokus erhält (z. B. durch Tabulator-Navigation).
  • :nth-child(n): Wählt das n-te Element in seiner übergeordneten Struktur aus.
  • :first-child: Wählt das erste Element in seiner übergeordneten Struktur aus.
  • :last-child: Wählt das letzte Element in seiner übergeordneten Struktur aus.
  • :nth-of-type(n): Wählt das n-te Elementtyp aus seiner übergeordneten Struktur aus.
  • :first-of-type: Wählt das erste Elementtyp in seiner übergeordneten Struktur aus.
  • :last-of-type: Wählt das letzte Elementtyp in seiner übergeordneten Struktur aus.
  • :not(selector): Wählt Elemente aus, die nicht dem angegebenen Selektor entsprechen.
  • :before: Fügt Inhalt vor dem Inhalt eines Elements hinzu.
  • :after: Fügt Inhalt nach dem Inhalt eines Elements hinzu.
  • :first-line: Wählt die erste Textzeile eines Elements aus.
  • :first-letter: Wählt den ersten Buchstaben eines Elements aus.
  • :enabled: Wählt aktivierbare Elemente aus (z. B. Eingabefelder).
  • :disabled: Wählt deaktivierte Elemente aus.
  • :checked: Wählt ausgewählte Elemente aus (z. B. bei Checkboxen).
  • :required: Wählt Elemente mit einem erforderlichen Attribut aus.
  • :valid: Wählt gültige Eingabeelemente aus.
  • :invalid: Wählt ungültige Eingabeelemente aus.
  • :nth-child(odd): Wählt ungerade Kinder aus.
  • :nth-child(even): Wählt gerade Kinder aus.
  • :lang(language): Wählt Elemente mit einer bestimmten Sprache aus.
  • :target: Wählt das Ziel-Element einer Anker-URL aus.
  • :root: Wählt das Wurzelelement des Dokuments aus.
  • ::before: Pseudo-Element, um Inhalt vor einem Element einzufügen.
  • ::after: Pseudo-Element, um Inhalt nach einem Element einzufügen.
  • ::first-line: Pseudo-Element, um die erste Zeile eines Elements zu formatieren.
  • ::first-letter: Pseudo-Element, um den ersten Buchstaben eines Elements zu formatieren.
  • ::selection: Pseudo-Element, um die Formatierung von ausgewähltem Text zu ändern.
  • @import: Bindet externe CSS-Dateien ein.
  • @font-face: Definiert benutzerdefinierte Schriftarten.
  • @page: Definiert das Layout für gedruckte Seiten.
  • @charset: Definiert das Zeichensatzkodierungsschema für CSS.
  • @supports: Prüft, ob der Browser eine bestimmte Eigenschaft unterstützt.
  • @keyframes: Definiert eine Animationssequenz.
  • @media: Definiert CSS-Regeln basierend auf dem Medientyp.
  • @namespace: Definiert einen XML-Namespace für eine CSS-Datei.
  • @-rule: Eine Regel für spezifische Browser-Engines.
