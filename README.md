Nutzung:
Erstelle ein neues virtual enviroment:
     $ python -m venv venv
    $ . venv/Scripts/activate
Installiere die benötigten python packages:
    $ pip install -r requirements.txt
Starte den Server:
    $ python wsgi.py
öffne im Browser die Website:
    http://127.0.0.1:5000/
